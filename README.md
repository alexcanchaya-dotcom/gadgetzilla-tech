# gadgetzilla-tech

GadgetZilla - A curated gadget discovery website featuring the best and latest tech gadgets.

## Stack
- Next.js App Router + TypeScript
- Tailwind CSS for the neon gaming aesthetic
- Framer Motion micro-interactions

## Getting Started
1. Install dependencies
   ```bash
   npm install
   ```
2. Run the development server
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to see the site.

The homepage features a neon hero, category filters, animated trending grid, and affiliate-ready CTAs tuned for gamers.
